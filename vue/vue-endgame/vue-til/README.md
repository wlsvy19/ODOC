# vue-til

## 뷰 프로젝트 생성
```
npm create vue-til
```

## vue cli(뷰명령어 먹히는거) 최신버전 설치
```
npm install -g @vue/cli
vue --version
```

## vue cli(뷰명령어 먹히는거) 특정 설치(프로젝트 진행은 이걸로)
```
npm install -g @vue/cli@4.5.11
vue --version
```

## vue-route 설치
```
npm install vue-route@3.5.3
```

## Project setup
```
npm install
```
