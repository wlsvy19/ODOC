# learn-nuxt  

# 최초 실행시 라이브러리 설치
`npm install`

# 넉스트 실행
learn-nuxt 폴더에서 터미널 열기
`npm run dev` 넉스트 서버 실행


# 백엔드 서버 실행
backend 폴더로 이동 후 터미널 열기(backend README.md 참고)
`npm run dev` 백엔드 서버 실행

## SSG 방식 배포-Netlify 사용
1. nuxt.config.js 에서 target을 static으로 설정
2. `npm run generate`
3. dist 폴더 생김()
