# Hacker News의 Open API를 사용하여 만든 vue-news 사이트


# 완성된 사이트 주소
https://leejp.netlify.app



## 뷰 프로젝트 생성
```
vue create vue-news
```

## vue 라우터 install
```
npm i vue-router@3.5.3 --save
```

## vuex 설치: 4버전x(에러남) vue2에는 vuex3버전 설치 해야함
```
npm i vuex@3.6.2
```

## Project setup
```
npm install
```
